# VocabuMe first cold launch kit

Date: 2026-05-04

## Goal

Get the first cold-user signal for VocabuMe without using the partner's existing contacts or private social capital.

Primary success signal:
- one non-friend opens the Mini App and adds a relocation scenario pack;
- better: one user reaches Premium paywall or starts Telegram Stars checkout;
- best: first paid Premium conversion.

## Positioning

Use the relocation wedge, not generic language learning.

Core sentence:

> VocabuMe is a practical phrasebook for moving to Georgia: bank, rent, Public Service Hall, residence/work documents, doctor, pharmacy, post, market, and first-week errands.

Important boundary:
- Do not present the product as legal advice.
- Do not promise that it explains the correct legal procedure.
- Say it helps prepare practical language for conversations with banks, landlords, clinics, pharmacies, Public Service Hall, and employers.

## Why now

Observed public pains:
- Georgia work authorization rules changed in 2026; public sources describe new work/right-to-work requirements and transition dates.
- Residence permit questions repeatedly involve Public Service Hall, documents, application status, translations, and follow-up.
- Expat banking questions recur around opening accounts as a non-resident, documents, cards, transfers, and bank terminology.
- Rent questions recur around viewing apartments, deposits, handover, utilities, building fees, and practical move-in details.

## Initial channels

Priority 1: Reddit
- `r/tbilisi`
- `r/Sakartvelo`
- relevant recent threads about bank accounts, residence permits, work permits, renting, first weeks in Tbilisi.

Reason:
- Public, searchable, cold audience.
- No need to use personal contacts.
- Direct linking is usually acceptable if the comment is useful and not spammy.

Rules:
- Reply only where the thread asks about an actual relocation problem.
- Lead with useful context, not the product.
- Mention the product once, softly, as a tool you built.
- Avoid repeated identical comments.

Initial target threads:
- `r/Sakartvelo`: "Georgia IE setup... is residence permit now mandatory too (on top of labor permit)?"
  Angle: answer with "not legal advice, but here are phrases to prepare for Public Service Hall / employer / application status".
- `r/Sakartvelo`: "IT residence permit. After getting this you should anyway receive the work permit additionally?"
  Angle: uncertainty around process; offer practical language prep, avoid interpreting law.
- `r/Sakartvelo`: "Work permit rules revised - fewer expats caught"
  Angle: confusing changes; useful phrases for checking status and asking about documents.
- `r/Sakartvelo` and `r/tbilisi`: "Opening a bank account as a foreigner"
  Angle: bank account opening phrases, documents, card, transfer, statement.
- `r/Sakartvelo`: "Anybody received work permit and work visa already?"
  Angle: application-status phrases and Public Service Hall questions.

Priority 2: public Telegram groups/channels
- Use only open/public groups or channels.
- If rules prohibit ads, do not post a promo.
- If rules are unclear, ask admin first.
- Good fit: answer a public question with useful phrases/checklist, then mention the Mini App only if it is naturally helpful.

Priority 3: web/Q&A/forum pages
- Only if a page allows comments and the answer is genuinely useful.
- Avoid low-quality SEO spam.

## First publishing sequence

1. Reddit soft comments:
   - find 3-5 recent relevant threads;
   - answer only if the thread asks about bank/residence/work/rent;
   - use one helpful paragraph plus link;
   - do not post the same text repeatedly.

2. Public Telegram:
   - identify 3 candidate public groups/channels;
   - check rules;
   - if ads are prohibited, do not post;
   - if rules are unclear, ask admin with the permission message.

3. Measure:
   - bot starts / Mini App opens;
   - pack additions;
   - paywall views;
   - checkout attempts;
   - paid conversions.

- Mini App opens from `https://t.me/VocabuMe_bot`.

- Starter packs are visible and addable.
